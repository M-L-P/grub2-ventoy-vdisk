<h1 align="center">grub2-ventoy-vdisk</h1>

启动安装在虚拟磁盘里的 Ventoy 并且拥有特殊的主题的 grub2，
- 拥有一个适应常见系列分辨率的解决方案；
- 没必要使用 U 盘；
- 没必要修改内置硬盘的 主引导记录 和 分区表；
- 对于碰巧手头没 U 盘 的用户来说，安装多系统将更加方便。
