<h1 align="center">grub2-ventoy-vdisk</h1>

啟動安裝在虛擬磁盤裏的 Ventoy 並且擁有特殊的主題的 grub2，
- 擁有一個適應常見系列分辨率的解決方案；
- 沒必要使用 U 盤；
- 沒必要把 .iso 文件 復製到 U 盤；
- 沒必要修改內置硬盤的 主引導記錄 和 分區表；
- 對於碰巧手頭沒 U 盤 的用戶來說，安裝多系統將更加方便。
